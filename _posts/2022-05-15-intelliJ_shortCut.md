- 액션 팁 보기 : Alt + Enter
- 최근에 작업한 파일 보기 : cmd + 
- 대/소문자 변환 : cmd + shift + U
  - 플러그인 CamelCase 사용시 alt + shift + U 로 카멜케이스, 스네이크, 케밥 등 다양하게 변환

- 원하는 곳 다중 커서 : Alt + Shift + 마우스 좌클릭
  - Alt 연타 후 위/아래 방향키 

- 프로젝트창 닫기 : Shitf + Esc
- 에러난 곳으로 바로 커서 이동 : F2
- cmd + P : 파라미터 정보 출력

- 테스트코드 자동 생성 : cmd + shift + T
